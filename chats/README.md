# תיקיית שיחות

תיקייה זו מכילה את כל השיחות שנשמרו במערכת.

## מבנה קובץ שיחה

כל שיחה נשמרת כקובץ JSON בפורמט הבא:

```json
{
  "chatId": "chat_1699876543210_abc123",
  "title": "מערכת ניהול משימות חכמה",
  "messages": [
    {
      "role": "user",
      "content": "אני רוצה לפתח מערכת..."
    },
    {
      "role": "assistant",
      "content": "מעולה! בואו נתחיל..."
    }
  ],
  "lastUpdated": "2025-11-13T10:30:00.000Z"
}
```

## שמות קבצים

- פורמט: `chat_[timestamp]_[random].json`
- דוגמה: `chat_1699876543210_abc123.json`

## גיבוי

להעתקת כל השיחות, פשוט העתק את כל התיקייה `chats/`

## ניקוי

למחיקת שיחות ישנות, מחק את הקבצים הרצויים או השתמש בממשק המנהל.

---

## ⚠️ חשוב ל-Production (Render)!

### נתיבי אחסון:
- **בפיתוח מקומי:** `./chats` (תיקיית הפרויקט)
- **ב-Render:** `/opt/render/project/src/chats` (על Persistent Disk)

### הגדרת Persistent Disk חובה!
אם לא תגדיר Persistent Disk ב-Render:
- ❌ השיחות ימחקו בכל הפעלה מחדש של השרת
- ❌ ממשק המנהל יהיה ריק
- ❌ כל הנתונים יאבדו

הוסף Disk בהגדרות Render:
- **Mount Path:** `/opt/render/project/src`
- **Size:** 1GB (חינמי)

---

**הערה:** קבצים בתיקייה זו נוצרים אוטומטית על ידי השרת.
