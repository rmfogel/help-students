# עוזר כתיבת הצעות פרויקט גמר 🎓

אתר אינטראקטיבי לעזרה בכתיבת הצעות לפרויקט גמר בהנדסת תוכנה ומדעי המחשב. האתר משתמש ב-ChatGPT API כדי לעזור לתלמידים להגדיר את הרעיון שלהם בצורה ברורה ומדויקת.

## תכונות ✨

- **צ'אט אינטראקטיבי** - שיחה חכמה עם מומחה AI שמבין את הצרכים שלך
- **שאלות מנחות** - המערכת שואלת שאלות ממוקדות שעוזרות להגדיר את הפרויקט
- **יצירת סעיפים** - עזרה בניסוח תיאור כללי והגדרת הבעיה האלגוריתמית
- **שמירה אוטומטית** - השיחה נשמרת אוטומטית
- **ייצוא לקובץ** - אפשרות לייצא את השיחה לקובץ טקסט
- **ממשק ניהול** - מנהל יכול לצפות בכל השיחות ולנהל את מפתח ה-API
- **עיצוב מינימליסטי** - ממשק נקי ופשוט
- **ממשק בעברית** - ממשק מלא בעברית עם תמיכה ב-RTL

## דרישות מקדימות 📋

- Node.js גרסה 14 ומעלה
- npm או yarn
- מפתח API של OpenAI

## התקנה והפעלה 🚀

1. **התקן תלויות:**
```powershell
npm install
```

2. **הפעל את השרת:**
```powershell
npm start
```

   או למצב פיתוח עם רענון אוטומטי:
```powershell
npm run dev
```

3. **פתח בדפדפן:**
   - ממשק משתמש: http://localhost:3000
   - ממשק מנהל: http://localhost:3000/admin.html

4. **הגדר מפתח API בממשק המנהל:**
   - היכנס לממשק המנהל (סיסמה ברירת מחדל: `admin123`)
   - הזן את מפתח ה-API שלך בהגדרות
   - לחץ "שמור מפתח"

## ממשק מנהל 👨‍💼

הממשק מאפשר למנהל:
- **צפייה בכל השיחות** - רשימה של כל הצ'אטים הפעילים עם כותרות
- **קריאת שיחות** - לחיצה על שיחה מציגה את כל ההתכתבות
- **מחיקת שיחות** - אפשרות למחוק שיחות
- **ניהול מפתח API** - הגדרת והחלפת מפתח ה-API
- **רענון אוטומטי** - הרשימה מתעדכנת אוטומטית כל 5 שניות

### גישה לממשק המנהל:
- כתובת: http://localhost:3000/admin.html
- סיסמה ברירת מחדל: `admin123` (שנה בקוד!)

## קבלת מפתח API 🔑

1. היכנס ל-[OpenAI Platform](https://platform.openai.com/)
2. צור חשבון או התחבר לחשבון קיים
3. עבור ל-[API Keys](https://platform.openai.com/api-keys)
4. לחץ על "Create new secret key"
5. העתק את המפתח והזן אותו בממשק המנהל

**שים לב:** השימוש ב-API של OpenAI כרוך בתשלום. וודא שיש לך קרדיט בחשבון.

## שמירת נתונים 💾

- **שיחות:** נשמרות בתיקייה `chats/` כקבצי JSON
- **מפתח API:** נשמר בקובץ `api_key.txt` 
- **פרומפט המערכת:** נשמר בקובץ `system_prompt.txt` (ניתן לעריכה!)
- **עמידות:** כל הנתונים נשמרים בדיסק ונטענים אוטומטית בהפעלה מחדש
- **גיבוי:** כל שיחה היא קובץ JSON נפרד שניתן לגבות

## אבטחה 🔒

- מפתח ה-API נשמר בקובץ מקומי בשרת (לא נחשף למשתמשים)
- ממשק המנהל מוגן בסיסמה
- קבצי השיחות ומפתח ה-API לא מועלים ל-Git
- **חשוב:** בסביבת ייצור, החלף את הסיסמה והצפן את הקבצים

## פתרון בעיות 🔧

### "מפתח API לא מוגדר"
- היכנס לממשק המנהל
- הזן את מפתח ה-API בהגדרות
- לחץ "שמור מפתח"

### "סיסמה שגויה" בממשק המנהל
- הסיסמה ברירת המחדל היא `admin123`
- אם שינית אותה, עדכן בקוד (`admin.js`)

### השרת לא עולה
- וודא ש-Node.js מותקן: `node --version`
- וודא שהתקנת את התלויות: `npm install`
- בדוק שהפורט 3000 פנוי

### השיחות נמחקות
- השיחות נשמרות בתיקייה `chats/`
- בדוק שהתיקייה קיימת ויש הרשאות כתיבה
- השיחות נשמרות אוטומטית בכל הודעה

### איפה הקבצים?
- **שיחות:** `chats/chat_[ID].json`
- **מפתח API:** `api_key.txt`
- **פרומפט המערכת:** `system_prompt.txt`
- הקבצים נוצרים אוטומטית בהרצה הראשונה

### איך לערוך את הפרומפט?
1. פתח את הקובץ `system_prompt.txt`
2. ערוך את הטקסט כרצונך
3. שמור את הקובץ
4. הפעל מחדש את השרת (`npm start`)
5. הפרומפט החדש ייטען אוטומטית
